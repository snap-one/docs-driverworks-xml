## searchtypes

`<search_types></search_types>`


### Parent

[`<devicedata>`][1]


This element contains the searchable text that is be used to locate the driver contained within the type XML element.


### Example

```xml
<search_types>
 <type>c4:av_gen</type>
</search_types>
```


[1]:	https://snap-one.github.io/docs-driverworks-xml/#devicedata

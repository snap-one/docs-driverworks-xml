## name

`<property></property>`


### Parent

`<property>`


The name element contains the name of the Property. This name is used to identify the property and is displayed  in Composer Pro.


### Example

```xml
<properties>
		<property>
			<name>Driver Version</name>
			<type>STRING</type>
			<readonly>true</readonly>
			<default>--</default>
		</property>
</properties>
```

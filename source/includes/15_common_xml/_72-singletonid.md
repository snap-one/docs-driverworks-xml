## control

`<SINGLETON_ID></SINGLETON_ID>`


### Parent

[`<config>`][1]


This element is used to ensures that the driver can only be loaded once within a project.

### Examples

```xml
<SINGLETON_ID>music_bridge_driver</SINGLETON_ID>
```
